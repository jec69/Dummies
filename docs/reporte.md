Reporte de Implementación y Pruebas

Nombre de la actividad

Implementación del módulo CRUD de usuarios y pruebas funcionales.

Objetivo

Desarrollar la estructura inicial de un CRUD de usuarios utilizando Node.js y Express, además de verificar su correcto funcionamiento mediante pruebas manuales.

Actividades realizadas

1. Se clonó el repositorio del proyecto.
2. Se creó la rama feature/crud-usuarios.
3. Se configuró el entorno de desarrollo con Node.js.
4. Se instaló la dependencia Express.
5. Se desarrolló el archivo app.js.
6. Se implementaron rutas básicas para las operaciones CRUD:
    * Consultar usuarios (GET)
    * Crear usuarios (POST)
    * Actualizar usuarios (PUT)
    * Eliminar usuarios (DELETE)
7. Se realizaron pruebas locales utilizando el navegador.
8. Se realizaron commits y push de los cambios.
9. Se creó una Pull Request para integrar los cambios al repositorio principal.

Resultados obtenidos

El servidor se ejecutó correctamente en el puerto 3000 y respondió adecuadamente a las solicitudes realizadas sobre la ruta /usuarios.

Las pruebas realizadas demostraron que la estructura inicial del CRUD funciona correctamente y que los cambios fueron integrados al flujo de trabajo mediante Git y GitHub.

Evidencias

* Ejecución correcta del servidor.
* Visualización de usuarios en localhost.
* Creación de rama de trabajo.
* Creación de commits.
* Creación de Pull Request.

Conclusión

Se logró implementar la estructura inicial del módulo CRUD de usuarios utilizando Node.js y Express. Además, se verificó el funcionamiento básico de la aplicación y se documentó el proceso mediante evidencias y control de versiones con GitHub.
